# Geometry

In this library, we calculate the perimeter and the area of the rectangle with length and breadth given.

## Build Instructions

Run the following command in the terminal
```
$ go build
```

## Test Instructions

Run the following command in the terminal
```
$ cd $PATH/geometry
$ go test
```
